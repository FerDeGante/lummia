# Lummia

Homepage premium para Lúmmia construida con Next.js App Router, Tailwind CSS v4, Framer Motion, GSAP + ScrollTrigger y Lenis.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Notas

- `public/lummia-logo.svg` es un placeholder temporal si el logo final aún no está disponible.
- `lib/constants.ts` contiene los datos y URLs fáciles de reemplazar.
